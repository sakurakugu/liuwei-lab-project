# 数码管引脚

| 引脚名称 | 功能        |
| -------- | ----------- |
| PIN1     | A1/B1/C1/D1 |
| PIN2     | A2/B2/C2/D2 |
| PIN3     | A3/B3/C3/D3 |
| PIN4     | E1/F1/G1/E2 |
| PIN5     | F2/G2/E3/F3 |

```
 ---A1      ---A2
|F1  |B1   |F2   |B2  上下摆头F3
 ---G1      ---G2     左右摆头E3
|E1  |C1   |E2   |C2  定时键    D3
 ---D1       ---D2
 冷风模式A3 暖风模式B3 开关C3
```
